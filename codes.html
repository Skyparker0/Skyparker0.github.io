All Ten

I helped my dad by writing a solver for the prototype of "all ten;" a math puzzle game where each day there is a different set of 4 numbers, which can be combined through mathematical operations to create the numbers 1-10. My python script discovered all solution sets to determine if the game could be hosted for a while, like wordle.
My source code is still hosted on the AoPS website, and the game gets on average 20,000+ users daily!

Try out the game!
